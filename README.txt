INSTRUCCIONES PARA SUBIR A GITHUB PAGES

1. Crear un repositorio en GitHub, por ejemplo: bl-consultores-web
2. Subir estos archivos en la raíz del repositorio:
   - index.html
   - styles.css
   - script.js
   - carpeta assets completa
3. Ir a Settings > Pages.
4. En Branch, seleccionar main y carpeta /root.
5. Guardar.
6. Esperar unos minutos y abrir la URL pública generada por GitHub Pages.

IMPORTANTE:
- El chatbot funciona dentro de la página y abre WhatsApp o correo cuando la persona solicita hablar con un asesor.
- GitHub Pages no envía mensajes automáticos por sí solo. Para formularios avanzados se puede usar Formspree, Google Forms o un backend externo.
